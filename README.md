    <h1>Custom AltusMetrum Clones</h1>
    <p>This repository is contains a few different custom AltusMetrum flight computer hardware design clones that I am working on. For more information please visit their <a href="https://altusmetrum.org">website</a>.</p>
  </div>
  <br>
  
  <div>
    <h2>Hardware:</h2>
    <p>In the Hardware/ folder you will find different KiCAD board designs. These boards are all clones of different AltusMetrum products. You can learn more about each one in their specific README files linked below.</p>
    <ul>
      <li>HardMega:</li>
      <p>Custom clone of the EasyMega V2 to use as a standalone staging computer in a 54mm 2 stage around an Aerotech 54mm forward closure. This is also my first attempt at an Altus clone.</p>
      <ul>
        <li><a href="Hardware/HardMega/README.MD">README</a></li>
        <li><a href="https://altusmetrum.org/EasyMega/">AltusMetrum EasyMega Information</a></li>
        <li><a href="https://git.gag.com/?p=hw/easymega;a=summary">AltusMetrum EasyMega Hardware Git</a></li>
      </ul>
    </ul>
  </div>
